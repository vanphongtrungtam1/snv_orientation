# Trang thông tin sinh hoạt đầu khóa — bản cập nhật v5

Bản này giữ nguyên các nội dung và chức năng hiện có, đồng thời cập nhật nhóm liên kết nhanh ở đầu trang:

- Đổi tên bốn nút thành **Thông tin chung**, **Hồ sơ thi**, **Lịch học chi tiết** và **Folder lớp học**.
- Giữ bốn nút trên cùng một hàng ở giao diện máy tính; trên màn hình rất hẹp, hàng nút có thể vuốt ngang để không bị xuống dòng.
- Mỗi nút sử dụng một màu riêng để học viên dễ phân biệt.
- Bổ sung mục **Lịch học chi tiết** vào thanh điều hướng phía trên, liên kết trực tiếp đến Google Sheets.
- Giữ nguyên hai nút hỏi đáp ẩn danh, khung quy định nghỉ tối đa **8/48 buổi**, nội dung và các chức năng còn lại.

## Các file cần tải lên GitHub

- `index.html`
- `style.css`
- `script.js`
- `logo-ctu.png`
- `logo-35.png`

Upload các file trên vào thư mục gốc của repository `snv_orientation` và chọn **Replace** khi GitHub hỏi có ghi đè file hay không.
